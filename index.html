what Alpha does

  • Creates pools — Uniswap V2, V3, V4 (hookless and hooked), Balancer V3
  • Manages liquidity — full-range positions, fee collection, rotation
  • Monitors prices — cross-pool arbitrage detection
  • Trades — swaps via V4 Clanker hooked pools, moltlaunch agent network
  • Stakes — RATCHET staking for yield

The codebase is at github.com/ABCDAO. The core library (@arbme/core-lib) handles all on-chain interactions.
